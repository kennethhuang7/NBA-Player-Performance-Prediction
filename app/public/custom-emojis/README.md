# Custom Emojis

Place your custom emoji images in this folder to use them in the chat.

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- GIF (for animated emojis)
- WebP

## Recommended Size
- 128x128 pixels or smaller
- Keep file sizes under 100KB for best performance

## Usage
1. Add your emoji image files to this folder
2. Name them descriptively (e.g., `happy_cat.png`, `thumbs_up.gif`)
3. They will automatically appear in the custom emoji section of the emoji picker

## Notes
- File names will be used as emoji identifiers
- Use lowercase and underscores for consistency (e.g., `my_emoji.png`)
- Avoid special characters in file names
